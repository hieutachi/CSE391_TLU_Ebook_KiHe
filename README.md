# CSE391 — Ebook tự học Frontend (Học kỳ hè)

Tài liệu hướng dẫn tự học môn **Nền tảng phát triển Web** (HTML · CSS · JavaScript · React) cho sinh viên TLU — bản **học kỳ hè** (`CSE391_TLU_Ebook_KiHe`).

## Nội dung

Thư mục [`ebook/`](./ebook/README.md):

- **12 buổi** lý thuyết–thực hành (đề tài xuyên suốt **CampusMart UI**)
- **12 phiếu học tập** (≥ 45 phút/phiếu): bài trên lớp + về nhà nối tiếp
- **Hướng dẫn Project** buổi 13–15 (nhóm)

**Hai mốc đầu ra của chuỗi phiếu:**

1. **Phiếu 08 (JS thuần):** một **Dashboard CampusMart phục vụ CRUD, có Form Thêm** (stats + lọc + danh sách + thêm sản phẩm).
2. **Phiếu 12 (đích cuối):** **ứng dụng React CRUD hoàn chỉnh** — 2 thực thể liên kết (Category → Product), đủ Thêm/Sửa/Xóa + quản lý danh mục. Không bắt buộc backend thật.

Dataset & hợp đồng chấm (chuẩn bị Autograder):  
[`ebook/worksheets/CANONICAL-DATA.md`](./ebook/worksheets/CANONICAL-DATA.md) · [`ebook/worksheets/GRADING-CONTRACT.md`](./ebook/worksheets/GRADING-CONTRACT.md)  
Tổng kho 8 SP CORE = **41380000** (cùng số liệu MiniShop Backend CSE485 — liên thông môn).

## Lưu ý cho sinh viên

- Đây là **hướng dẫn chi tiết + phiếu bài tập**, không phải source nghiệm thu để clone nộp.
- Mỗi phiếu: **tự viết code**, GitHub repo riêng, video OBS (có camera mặt).
- Làm lần lượt theo chuỗi; kết quả phiếu trước dùng lại ở phiếu sau.
- Không đổi SKU / giá / copy UI trong CANONICAL-DATA tùy tiện.
